# UK Housing Data Pipeline

## Project Overview

An end-to-end data engineering project using Python, PySpark, Databricks, BigQuery, SQL and Power BI to process, analyse and visualise UK housing market data.

## Tech Stack

- Python
- PySpark
- Databricks
- BigQuery
- SQL
- Power BI
- Git/GitHub

## Project Architecture

Raw Housing Data
→ Databricks (PySpark)
→ BigQuery
→ SQL Analysis
→ Power BI Dashboard
